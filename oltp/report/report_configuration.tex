\section{Platform Configuration}
In this section, hardware and software configuration of platform will be described. All tests were run in Docker\cite{docker} containers, considered as an operating-system-level virtualization service.

\subsection{Software}
\begin{itemize}
\item OS: Ubuntu Server 18.04.1 LTS
\item Kernel: Linux 4.15.0-29-generic
\item Docker: v18.06.1-ce
\item Postgres: Official Postgres v11.1 docker image
\item CockroachDB: Official CockroachDB v2.1.3 docker image
\end{itemize}

\subsection{Hardware}
\begin{itemize}
\item CPU: Intel(R) Xeon(R) E5530 @ 2.40 GHz
\item RAM: DIMM, 40 GB, DDR3, 1333 MHz
\item Storage: SCSI Disk, 4 TB, 15000 RPM
\end{itemize}
